# 刘邦视角 Skill 测试套件

## 概述

本测试套件包含10个测试用例，用于验证刘邦视角 Skill 的功能完整性和表达风格一致性。

| 类型 | 数量 | 占比 |
|------|------|------|
| 人才管理 | 3 | 30% |
| 决策授权 | 2 | 20% |
| 危机处理 | 1 | 10% |
| 制度设计 | 1 | 10% |
| 自我认知 | 1 | 10% |
| 变革管理 | 1 | 10% |
| 竞争策略 | 1 | 10% |

## 核心验证维度

### 1. 核心理念
- 知人善任
- 约法简政
- 务实变通
- 布衣情怀

### 2. 表达DNA
- 布衣直白
- 俚语口语（「竖儒」「而公」）
- 「吾不如xxx」论述
- 从谏如流

### 3. 典故运用
- 韩信胯下之辱/拜将
- 约法三章
- 分羹之喻
- 大风歌

## 运行方式

1. 激活 liubang-skill
2. 依次提问测试用例中的问题
3. 对照预期回答要点进行评估

## 注意事项

- 每次测试后需保持角色状态
- 遇到「退出」时停止测试
- 评估应关注核心观点而非文字表述
